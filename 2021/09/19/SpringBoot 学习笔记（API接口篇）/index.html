4）POST和GET的区别

GET POST
请求缓存 GET会被缓存 POST不会被缓存
用处 get常用于取回数据 post常用于提交数据
安全性 一般 比get安全
请求参数 get的querystring（仅支持urlencode编码 post的参数是放在body（支持多种编码）

这里仅简单罗列了一些区别，帮助读者理解，GET和POST还有许多不同，读者可自行搜索
